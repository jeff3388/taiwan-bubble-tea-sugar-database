# 貢獻指南（Contributing Guide）

## 最需要的貢獻

**有官方來源的品牌數據更新**：本資料庫部分數值來自食藥署抽查報告，而非所有品牌的最新官方公告。若您有更新的品牌官方營養標示，歡迎送出 PR。

**目前缺少的品牌**：
- 清心福全（Qing Xin）
- 大苑子（DaYuanzi）
- 日出茶太（Chatime，完整品項）
- 初鹿牧場系列
- 各連鎖超商（7-11、全家）自有飲品系列

## 貢獻規則

1. **每筆資料必須有來源**：必須是品牌官網、食藥署抽查報告或食品包裝標示截圖
2. **不接受估算值**：除非明確標示 `source: "estimated"` 並說明估算方法
3. **不加行銷語言**：不得寫「XX 品牌是最健康的選擇」類似文字
4. **id 格式**：`品牌縮寫-飲品名-糖度-容量ml`，全部 kebab-case

## 資料驗證

提交前請確認：
- `sugar_teaspoons = round(sugar_g / 4, 1)`
- `gl_estimated = round(gi_estimated × carbs_g / 100, 0)`
- `who_daily_sugar_pct = round(sugar_g / 50 × 100, 0)`
- `id` 在整個 drinks.json 中唯一

## 不接受的貢獻

- 品牌業配內容或比較推薦
- 沒有官方來源的數值
- 聲稱「某飲品健康/不健康」的主觀評語（資料欄位只描述事實）
